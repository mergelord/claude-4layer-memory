**Проблема:**
```
scripts/l4_semantic_global.py:413
Issue: [B110:try_except_pass] Try, Except, Pass detected.
Severity: Low   Confidence: High
```

**Рекомендация:**
Заменить `try-except-pass` на явную обработку ошибки с логированием:
```python
# Было:
try:
    collection.delete(ids=existing_ids)
except Exception:
    pass

# Должно быть:
try:
    collection.delete(ids=existing_ids)
except Exception as e:
    logger.warning(f"Could not delete existing embeddings: {e}")
```

**Вывод:** Очень безопасный код, критических уязвимостей не обнаружено.
