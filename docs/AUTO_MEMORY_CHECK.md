# Auto Memory Check - Автоматическая проверка памяти

**Версия:** 1.0.0  
**Дата:** 2026-04-19  
**Статус:** ✅ Активна

## Описание

Автоматическая проверка всех 4 слоёв памяти при завершении сессии. Гарантирует что важный контекст сохранён перед закрытием сессии Claude Code.

## Проблема

До внедрения этой фичи:
- Контекст мог теряться при завершении сессии
- Пользователь должен был вручную проверять все слои памяти
- Не было гарантии что важные решения сохранены

## Решение

Claude автоматически проверяет все 4 слоя памяти когда пользователь говорит о завершении сессии.

## Триггеры

Автоматическая проверка запускается при фразах:
- "завершаю сессию"
- "перезапускаю"
- "закрываю"
- "до встречи"
- "пока"
- "запомни контекст"
- "сохрани всё"

## Протокол проверки

### Layer 1: HOT (handoff.md)
```bash
Read: ~/.claude/projects/<project>/memory/handoff.md
```
- Проверяет наличие записей о текущей сессии
- Обычно пустой (обновляется через stop hook)

### Layer 2: WARM (decisions.md)
```bash
Read: ~/.claude/projects/<project>/memory/decisions.md
```
- Проверяет добавлены ли важные решения
- Если нет - добавляет автоматически

### Layer 3: COLD (MEMORY.md + файлы)
```bash
ls ~/.claude/projects/<project>/memory/*.md
Read: MEMORY.md
```
- Проверяет созданы ли новые файлы памяти
- Проверяет обновлён ли MEMORY.md
- Проверяет обновлён ли user_team.md с достижениями

### Layer 4: SEMANTIC (векторная БД)
```bash
python ~/.claude/hooks/l4_semantic_global.py stats
python ~/.claude/hooks/l4_semantic_global.py index-global
```
- Проверяет количество chunks (должно увеличиться)
- Переиндексирует если были изменения
- Проверяет семантический поиск

## Формат отчёта

```
## Проверка 4 слоёв памяти перед завершением сессии

✅ Layer 1 (HOT): Пустой (обновится через stop hook)
✅ Layer 2 (WARM): Обновлён - добавлено 2 решения
✅ Layer 3 (COLD): 3 файла обновлено
✅ Layer 4 (SEMANTIC): 126 chunks (+40 новых)

Контекст сохранён! Готово к завершению сессии.
```

## Предупреждения

Если что-то не сохранено:

```
⚠️ Обнаружены несохранённые данные:
- Layer 2: Важное решение о [X] не добавлено
- Layer 3: Достижения не обновлены

Сохранить перед завершением? (да/нет)
```
